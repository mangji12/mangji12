### 📌 주요 관심사 및 경험 (Focus)

**1. 배포 및 운영 자동화 경험**
- `FastAPI`와 `React`로 만든 서비스를 로컬에만 두지 않고, **AWS EC2**에 배포해 본 경험이 있습니다.
- **Jenkins**를 활용해 코드 푸시부터 배포까지의 과정을 자동화(CI/CD)하며, 개발 프로세스의 효율을 높이는 데 관심이 많습니다.

**2. 보안을 고려하는 개발 태도**
- 웹 해킹(Web Hacking) 학습을 통해 SQL Injection, XSS 등 주요 취약점의 원리를 이해하고 있습니다.
- 공격 방법을 아는 만큼, 방어할 수 있는 코드를 짜기 위해 입력값 검증이나 시큐어 코딩에 신경을 씁니다.

**3. 데이터 활용과 자동화**
- 공공데이터(한국소비자원)를 수집하고 가공하여 유의미한 정보를 제공하는 서비스를 개발 중입니다.
- 반복적인 작업을 줄이기 위해 커스텀 Admin을 만들거나 파이프라인을 구축하는 등, 개발 생산성을 높이는 도구 제작을 즐깁니다.
