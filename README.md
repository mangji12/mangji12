### 📌 주요 관심사 및 경험 (Focus)

**1. 배포 및 운영 자동화 경험**
- `FastAPI`와 `React`로 만든 서비스를 로컬에만 두지 않고, **AWS EC2**에 배포해 본 경험이 있습니다.
- **Jenkins**를 활용해 코드 푸시부터 배포까지의 과정을 자동화(CI/CD)하며, 개발 프로세스의 효율을 높이는 데 관심이 많습니다.

**2. 보안과 로우 레벨에 대한 이해**
- 웹 해킹(Web Hacking)뿐만 아니라 **포너블(Pwnable)**을 학습하며 메모리 구조(Stack, Heap)와 취약점의 근본 원리를 공부했습니다.
- 시스템의 작동 원리를 이해하고 코드를 작성하기에, 기능 구현을 넘어 **보안 구멍을 줄이는 개발**을 지향합니다.

**3. 데이터 활용과 자동화**
- 공공데이터(한국소비자원)를 수집하고 가공하여 유의미한 정보를 제공하는 서비스를 개발 중입니다.
- 반복적인 작업을 줄이기 위해 커스텀 Admin을 만들거나 파이프라인을 구축하는 등, 생산성을 높이는 도구 제작을 즐깁니다.
